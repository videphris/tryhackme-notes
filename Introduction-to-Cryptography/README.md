### Core Concept (In My Own Words)
--- Explain the main idea simply and clearly
Symmetric key encryption is used for efficient and secure communication utilizing the same key to decrypt and encrypt.
Asymmetric key encryption is best used to set up symmetric key encryption by having both a public and private key for encryption and decryption. For example, if a file is encrypted with the public key it can only be decrypted with the private key and vice versa. If it not possible to obtain the private key from knowing someone's public key.
